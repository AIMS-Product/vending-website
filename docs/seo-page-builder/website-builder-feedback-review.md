Status legend:

- Done: implemented end-to-end enough to satisfy the feedback.
- Partial: data model or some UI exists, but the requested workflow is incomplete.

- Needs product call: feedback is broader than the current builder scope or needs prioritisation.

Review caveat: this pass is based on repo inspection, not a fresh browser walkthrough. UI-facing statuses should be rechecked in the rendered admin before final sign-off.

## Highest-Leverage Remaining Gaps

1. Page templates and page type workflow: currently disabled in code, and it blocks several feedback items at once.
2. Page-level bulk operations and metadata editing: list page exists, but bulk page actions/meta edits are missing.
3. General redirect manager: redirect infrastructure exists, but the admin management surface is missing.
4. Collaboration layer: presence, comments/annotations, and overwrite conflict detection are not implemented.
5. Lifecycle automation: scheduled publishing, expiry/review reminders, and A/B testing are not present.
