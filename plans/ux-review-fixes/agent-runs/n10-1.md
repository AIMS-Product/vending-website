## CLS numbers + shift-source analysis

Measured with a buffered `layout-shift` PerformanceObserver in Playwright
(viewport 1280×800, networkidle + 2.5s settle), **dev mode**:

- `/` → **CLS 0.0000** (the reviewer's 0.263 on `/` did NOT reproduce here; likely
  a cold-compile / warm-cache difference in dev).
- `/news` → **CLS 0.2625**, from a **single** shift at ~436ms.

**Shift source (fixed vs reported):**

- The only shift source is the **`<footer>`** (`FOOTER.border-t-2...bg-[#f5fbff]`),
  moving from `y=590 h=210` to `y=0 h=0`. This is the footer being repositioned as
  the page content above it streams/hydrates in dev — a single early shift, not a
  per-image reflow.
- **None of the news-card images shifted.** The card image box already reserves
  space via `aspect-[16/9]` + `fill` + `sizes`, so no width/height fix was needed
  and none was applied (correctly within scope).
- I confirmed the candidate culprits above the footer — `FinalCta` (on /news) uses
  `aspect-[4/3]`/`aspect-auto` + `fill`, and NewsHero/NewsList reserve their boxes —
  so this is **not** a missing-image-dimension bug in any component. The footer
  shift is a dev-mode streaming/hydration artifact.
- **REPORTED, not fixed (out of scope):** the footer shift is site chrome and the
  root cause is page streaming, not an image dimension within my write scope. It
  should be re-judged against a **production build** (final-proof node re-measures);
  authoritative CLS judgement requires `next build`, which dev numbers overstate.

**Separate out-of-scope observation (REPORTED):** under the forced-failure test,
three non-card images break and show raw alt text because they don't use a fallback
wrapper — the Wordmark logo in `<header>` and `<footer>`
(`src/components/site/Wordmark.tsx`) and the FinalCta section image
(`src/components/sections/FinalCta.tsx`, alt "A customer tapping a phone to pay at
a smart vending machine"). These are outside N10's write scope (site chrome / N9
sections). If broader image-fallback coverage is wanted, `ImageWithFallback` is now
available to drop into those components.
